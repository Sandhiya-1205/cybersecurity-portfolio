Case Studies

LIVE ENGAGEMENT WEB APPLICATION VAPT

HindustanSchools.in
Security Assessment Report

A structured, methodology-driven web application penetration test against a live education platform. Identified and documented critical vulnerabilities using the OWASP Top 10 framework, manual testing techniques, and automated tooling — culminating in a boardroom-ready security report with actionable remediation guidance.

01
Reconnaissance

Passive OSINT via WHOIS, Shodan, and Google dorking. Active recon with Nmap enumerated open ports (80, 443, 3306), service versions, OS fingerprinting, and CMS identification.

02
Scanning & Enumeration

Gobuster bruteforced directories, exposing /admin and /backup. Burp Suite intercepted session cookies, auth tokens, and POST parameters. OWASP ZAP ran full DAST scans across all endpoints.

03
Exploitation & Findings

Confirmed SQL Injection in login endpoint via SQLmap. Identified IDOR at /user?id=, reflected XSS in search param, and missing HSTS/CSP headers. All findings CVSS v3.1 scored.

04
Risk Report Delivery

Delivered structured VAPT report with executive summary, technical evidence with PoC screenshots, risk matrix, CVSS scoring, and a prioritised remediation roadmap aligned with OWASP.
